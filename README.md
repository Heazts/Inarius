# Inarius v0.0.1 Beta

O Inarius é um sistema de consulta focado na base de dados da ABCFarma (edição de Junho de 2026).

> [!WARNING]
> **Status do Projeto: VERSÃO BETA**
> Este software está atualmente em fase Beta de desenvolvimento e testes. O sistema ainda apresenta falhas, erros de OCR e imprecisões no motor de pesquisa. Os resultados exibidos podem não estar perfeitamente formatados ou não refletir 100% o documento original devido às limitações momentâneas de extração.

## Funcionalidades Atuais
- **Visualização de PDF**: Integração da página do catálogo atrelada ao resultado da busca.
- **UX**: Design minimalista em temas High-Contrast.
- **Favoritos**: Salvamento temporário de itens em cache local.

## Origem dos Dados
Todos os dados estruturados neste projeto refletem o repositório da ABCFarma - Junho de 2026 (sujeito a erros de transcrição da versão Beta).
