# Inarius v0.0.1 Beta

O Inarius é um sistema de consulta focado na base de dados da ABCFarma (ediçăo de Junho de 2026).

> [!WARNING]
> **Status do Projeto: VERSĂO BETA**
> Este software está atualmente em fase Beta de desenvolvimento e testes. O sistema ainda apresenta falhas, erros de OCR e imprecisőes no motor de pesquisa. Os resultados exibidos podem năo estar perfeitamente formatados ou năo refletir 100% o documento original devido ŕs limitaçőes momentâneas de extraçăo.

## Funcionalidades Atuais
- **Visualizaçăo de PDF**: Integraçăo da página do catálogo atrelada ao resultado da busca.
- **UX**: Design minimalista em temas High-Contrast.
- **Favoritos**: Salvamento temporário de itens em cache local.

## Origem dos Dados
Todos os dados estruturados neste projeto refletem o repositório da ABCFarma - Junho de 2026 (sujeito a erros de transcriçăo da versăo Beta).
